# LazyVim

Install in WSL2 AlmaLinux:

```bash
git clone https://github.com/LazyVim/starter ~/.config/nvim
rm -rf ~/.config/nvim/.git
nvim
```

Recommended external tools:

```bash
sudo dnf install -y ripgrep fd-find fzf gcc gcc-c++ make cmake ninja-build gettext unzip
npm install -g neovim prettier eslint_d typescript typescript-language-server
pipx install black
pipx install ruff
```
